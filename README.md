## Features
- Kotlin
- MVVM Pattern
- Minimum SDK level 21
- Retrofit2 - HTTP Client
- OkHttp - Network interceptor
- Glide - Loading images
- Gson - Json library
- Jetpack
  - LiveData
  - ViewModel
  - Databinding
- Kakao SDK - User login
- AWS S3 SDK - Upload images

## CI/CD
- Github Actions
- InAppUpdate
